# Project Scope

The Vehicle Tracking & Counting project demonstrates a beginner-level Artificial Intelligence application that stores and analyzes vehicle count data. It provides a summary of different vehicle types and visualizes the results using a bar chart.

---

# Project Objective

- Count different types of vehicles.
- Display a vehicle count report.
- Visualize vehicle data using graphs.
- Demonstrate basic AI and data analysis concepts.

---

# Technologies Used

- Python
- Pandas
- Matplotlib

---

# Dataset

The project uses a CSV dataset named **vehicles.csv** containing vehicle types and their counts.

---

# Working

1. Load the dataset.
2. Read vehicle counts.
3. Display the count of each vehicle type.
4. Calculate the total number of vehicles.
5. Generate a bar chart.
